## Description
This project aims at building a simple yet usefull clock for kids that could halp them get organized.
Functions:
* Display time and date
* Display weather forecast
* Display next activities
* Alarm depending on day of the week
* brightness 

Using HUB75 LED Matrix Panel (64x32)

Based on Arduino projects by Tech Talkies YouTube Channel.
https://www.youtube.com/@techtalkies1
https://techtalkies.in/2024/07/19/rgb-internet-clock-with-esp32/?i=1
